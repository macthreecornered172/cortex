## 🚀 What is cortex?

cortex is a desktop app for running multi-agent workflows on your computer. It helps you set up tasks, connect agents, and watch them work in real time. It also gives you a live dashboard so you can track what is happening without opening logs or command windows.

It is built for local use, with support for Docker and Kubernetes setups too. If you want a clear view of agent work, cortex keeps the process in one place.

## 🔧 How cortex works

cortex uses a workflow graph. That means you can connect steps in a clear order, then let the app manage the run. It also supports mesh mode, which lets agents work as a group instead of in a simple line.

Main parts you may see:

- Workflows: the full task flow
- Agents: the workers that handle tasks
- Nodes: the steps in a workflow
- Dashboard: the live view of what is running
- Logs: text records of events and actions

You do not need to know the terms in depth to use the app. The dashboard helps make the process easy to follow.

## 🛠️ Troubleshooting

If the app does not open:

1. Right-click the file and run it as admin.
2. Check that Windows did not block the download.
3. Re-download the file from the releases page.
4. Make sure your antivirus did not remove part of the app.

If the dashboard does not load:

1. Close the app.
2. Open it again.
3. Check that the local service is running.
4. Make sure no other app is using the same port.

If your workflow fails to start:

1. Review the workflow setup.
2. Check node names and links.
3. Confirm the agent files are in the right place.
4. Try a simpler workflow first.
